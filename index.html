About

This sidecar.xml generator is brought to you by Johannes Henseler, author of the blog Secrets and Happiness of Digital Publishing".

I made this generator mainly for myself, but recently rebuilt everything from scratch using twitter/bootstrap. I checked everything, but I cannot guarantee for your sidecar file to work.

If you have any questions or issues, feel free to contact me directly at frisch@frischmil.ch, or tweet me @frischmilch.

Project Details

This project and it's code is available on github where you can download the source code, fork it or put in any issues.
